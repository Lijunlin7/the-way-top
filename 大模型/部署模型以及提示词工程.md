### **如何查看 Python 版本并确定下载的 Wheel 文件**

#### **1. 查看 Python 版本**

在终端或命令行中运行以下命令：

```
python --version
# 或
python3 --version
```

```
Python 3.9.13
```

**关键信息**：

- **主版本号**（如 `3.9`）
- **次版本号**（如 `.13`）
- **位数**（32位或64位，通常64位系统会显示 `AMD64` 或 `x86_64`）

#### **2. 检查 Python 解释器位数**

运行以下代码查看系统是 **32位** 还是 **64位**：

```
python -c "import sys; print(sys.maxsize > 2**32)"
```

- 输出 `True`：64位 Python
- 输出 `False`：32位 Python

#### **3. 确定 Wheel 文件名格式**

Wheel 文件名通常遵循以下格式：

```
包名-版本号-cpPython版本-abi标签-平台标签.whl
```

```
llama_cpp_python-0.2.23-cp39-cp39-win_amd64.whl
```

- **`cp39`**：适用于 Python 3.9
- **`win_amd64`**：64位 Windows 系统
- **`manylinux`**：Linux 系统
- **`macosx`**：macOS 系统

#### **4. 如何选择正确的 Wheel 文件**

1. **匹配 Python 版本**
   - 如果你的 Python 是 `3.9.x`，选择 `cp39` 的 Wheel。
   - 如果是 `3.10.x`，选择 `cp310`，以此类推。
2. **匹配操作系统和位数**
   - **Windows**：
     - `win32` → 32位
     - `win_amd64` → 64位
   - **Linux**：
     - `manylinux_x86_64` → 64位
     - `manylinux_i686` → 32位
   - **macOS**：
     - `macosx_10_9_x86_64` → Intel 芯片
     - `macosx_11_0_arm64` → Apple Silicon (M1/M2)
3. **ABI 兼容性**
   - 通常选择 `abi3`（兼容大多数情况）或 `none`（无特殊依赖）。
